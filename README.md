# konstantinaporfyri-site

Επαγγελματική ιστοσελίδα για την Κωνσταντίνα Πορφύρη — Συμβουλευτική Ψυχολόγος & Συνθετική Ψυχοθεραπεύτρια.

## Περιγραφή
Στατική σελίδα μίας στήλης (single-page), φιλοξενούμενη στο GitHub Pages.

- **`index.html`** — όλο το περιεχόμενο σε ένα αρχείο (semantic HTML, SEO meta, JSON-LD).
- **`style.css`** — θεματική σε αποχρώσεις magenta, responsive, με υποστήριξη προσβασιμότητας.
- **`images/`** — λογότυπο, φωτογραφία προφίλ, εικόνες λουλουδιών/θεραπείας.
- **Custom domain:** konstantinaporfyri.gr

## Χαρακτηριστικά
- Hero με ανθική εικόνα, κάθετη πλοήγηση που «κολλάει» (sticky nav) με mobile menu.
- Ενότητες: Σχετικά με μένα, Εργασιακό ιστορικό (timeline), Εκπαίδευση, Σεμινάρια, Επικοινωνία.
- **Προσβασιμότητα:** skip link, σωστή ιεραρχία επικεφαλίδων, μενού με μεγέθυνση γραμματοσειράς,
  λειτουργία υψηλής αντίθεσης, υπογράμμιση συνδέσμων (αποθηκεύονται στον browser),
  ορατό focus, υποστήριξη `prefers-reduced-motion`.

## Τοπική προεπισκόπηση
Επειδή το περιεχόμενο είναι πλέον inline, αρκεί να ανοίξετε το `index.html`. Για πλήρη λειτουργία:

```bash
python3 -m http.server 8000
# http://localhost:8000
```

## Σημείωση
Ο φάκελος `content/` (παλιά αρχεία που φορτώνονταν δυναμικά) δεν χρησιμοποιείται πλέον —
το περιεχόμενο ενσωματώθηκε στο `index.html`. Μπορεί να αφαιρεθεί με ασφάλεια.
